Introducing Pico.

A stupidly simple & blazing fast, flat CMS.
Making the web easy.

Stupidly Simple

Pico makes creating and maintaining a
website as simple as editing text files.

Blazing Fast

Pico is seriously lightweight and
doesn't use a database, making it super fast.

No Database

Pico is a "flat file" CMS, meaning no
database woes, no MySQL queries, nothing.

Markdown Formatting

Edit for website in your favorite text
editor using simple Markdown formatting.

Twig Templates

Pico uses the Twig templating engine,
for power and flexible themes.

Open Source

Pico is completely free and open source,
released under the MIT license.

Want to contribute?

Help make Pico better by checking out the Github repository and submitting pull requests.
If you find a bug, please report it on our issues page.
